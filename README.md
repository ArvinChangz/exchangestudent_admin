# 交換生申請系統（管理者端） Exchange Student Application System

為簡易交換生統管理者介面端

使用JavaScript、HTML、CSS撰寫

## 功能

* 申請資料查詢
* 帳號管理
* 交換學校管理
* 審查設定
* 修改Log
* 開放時間設定
